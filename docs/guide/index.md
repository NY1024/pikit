# Guides

pikit's methods are organized into four orthogonal dimensions. Each guide page
covers one dimension in depth — what it does, how to use it, and what each
method produces.

| Guide | Dimension | What it controls |
|---|---|---|
| [Attacks](attacks.md) | `attack` | How the payload is *worded* |
| [Defenses](defenses.md) | `defense` | How the prompt is *hardened* |
| [Channels](channels.md) | `channel` | Where the payload is *hidden* (indirect) |
| [Agents](agents.md) | `agent` | What *receives* the attack |
| [Targets](targets.md) | `target` | Which model *backend* to use |
| [craft() API](craft.md) | — | The unified entry point |
